### AWS CLI and SDKs

**AWS CLI:**
```bash
# Install AWS CLI
pip install awscli

# Configure
aws configure

# Glue commands
aws glue get-databases
aws glue get-tables --database-name my_db
aws glue start-crawler --name my-crawler

# Lake Formation commands
aws lakeformation list-permissions
aws lakeformation grant-permissions
```

**Boto3 (Python SDK):**
```python
import boto3

# Glue client
glue = boto3.client('glue')
response = glue.get_table(DatabaseName='my_db', Name='my_table')

# Lake Formation client
lakeformation = boto3.client('lakeformation')
response = lakeformation.list_permissions()
```

**AWS SDK for Java:**
```java
GlueClient glue = GlueClient.builder().region(Region.US_EAST_1).build();
GetTableRequest request = GetTableRequest.builder()
    .databaseName("my_db")
    .name("my_table")
    .build();
GetTableResponse response = glue.getTable(request);
```
